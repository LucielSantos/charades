Crie um app que seja um jogo de mímica.
O app deve ser um jogo de mímica, onde o usuário deve adivinhar a palavra que está sendo mostrada na tela.
Requisitos:
Crud de times
Categorias: Animais, Profissões, Biblia, Filmes/Series, Ações, Personagens (Herois, Viloes, desenhos, etc). Categorias devem ser pré cadastradas no código, porém em um futuro deve ser possível adicionar novas categorias e palavras.
Deve ser possível selecionar quais categorias serão usadas no jogo.
Sistema de dificuldade: Fácil, Médio, Difícil.
Sistema de tempo: Cronometro, Tempo ilimitado.
Nos Ultimos 10 Segundos adicionar adicionar efeito de "pressão"
deve ser possível: esconder palavra, pular, acertar, errar.
Ao validar a palavra, deve ser mostrado algum tipo de feedback para o usuário.
Ranking de time
Cada time deve ter um nome, uma cor.
Não precisa de login, deve salvar os dados no localStorage, porém no futuro deve ser possível salvar os dados em um banco de dados.
Responsivo, Mobile first.
multi language, porém inicialmente apenas em português.
Preparação para PWA e SEO.
O app deve ser feito em Next JS com Typescript (Biome), Tailwind CSS, Shadcn UI, Lucid, se achar necessario uma lib de gerenciamento de estado usar alguma que seja popular e bem suportada, outras libs voce decide caso precise.
